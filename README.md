# nexus--plateform
A complete collaboration platform where investors and entrepreneurs can schedule meetings, sign documents, make video calls, manage payments, and secure their accounts.
📌 Project Title
Nexus Platform - Investor & Entrepreneur Collaboration Platform

📌 Project Overview
A complete collaboration platform where investors and entrepreneurs can schedule meetings, sign documents, make video calls, manage payments, and secure their accounts.

📌 Features Completed
Week 1: Calendar & Scheduling
✅ Interactive calendar with date picker

✅ Add availability slots with start/end time

✅ View all available meeting slots

✅ Delete slots

✅ LocalStorage persistence (data saves automatically)

Week 2: Documents & Video Calling
✅ Document Chamber with E-signature

✅ Investment agreement template

✅ One-click document signing

✅ Video call with real camera/mic access

✅ Start/End call functionality

✅ Mute/Unmute audio

✅ Turn on/off video

Week 3: Payments & Security
✅ Payment wallet with balance display

✅ Deposit money

✅ Withdraw money

✅ Transaction history with timestamps

✅ Password strength meter

✅ Two-Factor Authentication (2FA) mock (OTP: 123456)

✅ Role-based UI (Investor/Entrepreneur)

📌 Technical Details
Technology	Purpose
React.js	Frontend framework
Vite	Build tool
React Calendar	Calendar component
WebRTC	Video calling
LocalStorage	Data persistence
CSS-in-JS	Styling
📌 How to Use
Calendar Tab

Click any date → Add Slot → Select time → Save

View all slots in right panel

Delete unwanted slots

Documents Tab

Read agreement

Click "Sign Document"

Document status changes to SIGNED

Video Call Tab

Click "Start Call"

Allow camera permission

Use video/audio controls

Click "End Call" to stop

Payment Tab

Enter amount

Click Deposit or Withdraw

View transaction history below

Security Tab

Type password to see strength meter

Click "Enable 2FA"

Enter OTP: 123456 to verify
